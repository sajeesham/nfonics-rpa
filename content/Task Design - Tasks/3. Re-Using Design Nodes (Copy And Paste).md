To reuse the jiffy nodes from existing tasks, users can copy and paste the Jiffy nodes from existing tasks to new tasks. 

Steps to be followed,

 * Open existing task
 + Press CTRL key+ Select the nodes
 + Copy the nodes (Ctrl +C)
 + Open new task
 + Paste (CTRL +V)
